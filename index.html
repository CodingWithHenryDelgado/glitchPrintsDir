What's The Right Strength For Me?

For most “standard” prints that don’t need to be super strong, we suggest using an infill density of 15-50%. This density percentage provides okay strength.

Functional prints need to be strong. Therefore, we recommend using a higher infill: of more than 50%. The payoff will be a stronger and heavier part.

Consider using an infill density of 0-15% for small figurine models meant for display purposes only. Models printed with this density range will be lightweight and not very strong.

Keep in mind, however, that the higher the infill density, the less flexible the part will be.

* All information provided from All3DP.

What's The Right Quality For Me?

If the print has multiple parts that attach or has some details, you would like to keep; I suggest using standard quality.

If you want all those little details printed for a sculpture like print, then I recommend Full HD. (Please note that Full HD will take more time, so there will be a fee.)

You can use the draft quality for prints that don't need details or are low poly.
